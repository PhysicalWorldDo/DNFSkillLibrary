# DNFSkillLibrary - DNF全职业技能对照表

地下城与勇士（DNF）全职业技能资源浏览工具，提供可视化的技能贴图预览和资源管理功能。

## 功能概览

### 职业技能浏览
- 覆盖 DNF 全部 17 个职业（男鬼剑士、女魔法师、男神枪手、暗夜使者、枪剑士、弓箭手等）
- 树形结构按职业 → 技能层级组织，快速定位目标技能
- 支持「所有技能」和「已找好的技能」两种视图切换

### 贴图实时预览
- 点击任意 IMG 路径即可从 NPK 文件中解码并显示贴图
- 帧浏览：逐帧前进/后退，查看动画每一帧的细节
- 自动播放：可调节 FPS（1-60），循环播放动画序列
- 状态栏显示当前帧号、总帧数、贴图尺寸

### 中英文切换
- 一键切换技能名和职业名的中英文显示
- 中文翻译数据按职业独立存储（skill_name_translations/ 目录）
- 保留英文原名作为 tooltip 提示

### 路径筛选
- 支持全局屏蔽不需要的 IMG 路径，屏蔽后所有技能列表自动过滤
- 已屏蔽路径面板可查看和批量恢复

### 资源导出
- **导出 NPK**：将当前技能的所有可见 IMG 路径打包为独立 NPK 文件，支持进度条和取消
- **导出 JSON**：将路径列表导出为标准 JSON 格式，方便后续处理

## 使用说明

### 运行
双击 `DNF全职业技能对照表.exe` 启动程序。

### 基本操作
1. 点击「浏览」选择 NPK 文件所在的根目录
2. 在左侧树形列表中选择职业和技能
3. 中栏显示该技能的所有 IMG 路径，点击即可在右侧预览
4. 使用底部帧导航或播放按钮浏览动画

### 文件结构
```
DNFSkillLibrary/
├── DNF全职业技能对照表.exe    # 主程序
├── dnflib.dll                 # NPK/IMG 解析库
├── README.md
├── LICENSE
├── resource_paths_skillreplay/ # 全部技能资源路径（按职业/技能目录组织）
├── resource_paths_found/       # 已整理确认的技能资源
├── skill_name_translations/    # 技能名中英文翻译（每职业一个 JSON）
└── filtered_paths.json         # 屏蔽路径配置（程序自动管理）
```

## 数据格式

每个技能对应一个 JSON 文件，格式如下：

```json
{
  "skill": "SkillName",
  "resource_paths": [
    "sprite/character/xxx/effect/skillname.img",
    ...
  ]
}
```
